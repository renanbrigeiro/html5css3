Introdução ao GIT
Comandos básicos:

Passo 1: Configurações iniciais
git config --global user.name "seu nome"
git config --global user.email "seu email"

Passo 2: Clonar um repositório
git clone (link do repositório)

Passo 3: Administrando o repositório local
git status (verificar o status do repositório)
git add .  (adicionar o conteúdo do projeto no "container")
git commit -m "descrição" (documenta e armazena o container)

Passo 4: Sincronizar as mudanção com o GITHUB
git push
